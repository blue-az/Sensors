# Project Title

BabHist

## Description

Wrangle Bab data from BabExt.db which was extracted from Bab Pop txt file, first to JSON, then to .db. Extracted fields are time (unix posix time needs to divided by 10000), type, spin, score and value for each of: Style, Effect, Speed. Also, added stroke counter and session counter during extraction. Data appears to be collected between 4/27/23 and 9/23, but also appears to have 1890(!) sessions according to the generated counter. All code has been generated by GenAI.
